# TimeTracker Pro

Eine moderne Webanwendung zur präzisen Erfassung von Arbeitszeiten, Pausen und automatischer Pausenberechnung. Integriert mit Supabase für Cloud-Synchronisierung und Authentifizierung.

## Tech Stack
- **Frontend**: React + Vite
- **Styling**: Vanilla CSS (Glassmorphism)
- **Database**: Supabase
- **Icons**: Lucide React
- **Charts**: Chart.js
- **Exports**: Excel (XLSX), PDF (jsPDF)
